# Proyecto: Aplicación Web de Gestión de Vinilos

Este proyecto consiste en una aplicación web diseñada para gestionar colecciones personales de discos de vinilo. Permite a los usuarios registrarse, iniciar sesión, realizar búsquedas manuales de discos mediante APIs externas y guardar sus propios discos en una colección privada. El objetivo principal es ofrecer una herramienta sencilla, moderna y funcional para catalogar música de forma personalizada.

La aplicación está desarrollada por tres estudiantes durante un periodo de tres meses (abril–junio), combinando aprendizaje y desarrollo real. El trabajo se divide en tres áreas principales: interfaz web, servidor y base de datos con integraciones externas.

## Tecnologías utilizadas
- **Frontend:** React + Vite  
- **Lenguaje del frontend:** JavaScript + JSX  
- **Estilos:** CSS y/o TailwindCSS  
- **Backend:** Java + Spring Boot  
- **Base de datos:** MySQL  
- **APIs externas:** Discogs, MusicBrainz, Google Vision API, ACRCloud  

## Funcionalidades principales
- Registro e inicio de sesión de usuarios con autenticación mediante tokens.  
- Búsqueda manual de discos utilizando APIs externas.  
- Visualización de resultados con información detallada de cada disco.  
- Gestión de una colección personal: añadir, listar y filtrar discos.  
- Conexión completa entre interfaz web, servidor y base de datos.

## Organización del equipo
- **Interfaz web:** desarrollo de pantallas, componentes, diseño y conexión con el servidor.  
- **Servidor:** creación de endpoints, lógica de negocio, autenticación y seguridad.  
- **Base de datos y APIs:** diseño de tablas, relaciones, consultas y comunicación con servicios externos.

## Objetivo del MVP
El objetivo del Producto Mínimo Viable es disponer de una aplicación funcional que permita:
- Crear cuentas de usuario.  
- Iniciar sesión de forma segura.  
- Buscar discos manualmente.  
- Consultar información detallada de cada disco.  
- Guardar discos en una colección personal.
